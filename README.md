# 📊 지역별 연령별 인구구조 시각화 웹앱

행정안전부 "연령별 인구현황" 데이터를 기반으로, 지역(시도/시군구/읍면동)별 인구 피라미드 구조를 시각화하고
전국에서 인구구조가 가장 비슷한 지역을 찾아주는 Streamlit 웹앱입니다.

## 🔗 배포

Streamlit Community Cloud를 통해 배포합니다. ([Streamlit Cloud](https://streamlit.io/cloud))

## ✨ 주요 기능

### 1. 지역별 연령 구조 비교
- 지역명 검색(텍스트 입력) + 드롭다운 선택을 조합한 지역 검색
- 여러 지역을 동시에 선택해 인구구조를 하나의 꺾은선 그래프로 비교
- 성별(계/남/여) 필터
- Plotly 기반 인터랙티브 그래프 (hover, zoom 등 지원)

### 2. 전국 유사 지역 Top N 탐색
- 기준 지역(읍면동 포함)을 검색·선택하면, 연령대별 인구 **비율(구성비)** 기준으로
  유클리드 거리가 가장 가까운 지역 Top N(3~10개, 슬라이더 조절)을 탐색
- 지역 규모(인구수) 차이가 아닌 **인구 피라미드의 형태 유사성**을 비교
- 기준 지역은 굵은 실선, 유사 지역은 점선으로 겹쳐 표시하여 비교

### 3. 원본 데이터 확인
- 선택한 지역의 원본 인구 수치를 표 형태로 확인 가능

## 🚀 로컬 실행 방법

```bash
# 1. 저장소 클론
git clone <이 저장소의 URL>
cd <저장소 폴더>

# 2. 패키지 설치
pip install -r requirements.txt

# 3. 데이터 파일을 app.py와 같은 폴더에 위치시킨 뒤 실행
streamlit run app.py
```

## ☁️ Streamlit Community Cloud 배포 방법

1. `app.py`, `requirements.txt`, 데이터 CSV 파일을 GitHub 저장소에 push
2. [share.streamlit.io](https://share.streamlit.io) 접속 후 GitHub 계정 연동
3. "New app" → 해당 저장소 선택 → Main file path에 `app.py` 지정
4. Deploy 클릭

## 📌 데이터 및 분석 방법 참고

- 원본 데이터는 전국 → 시도 → 시군구 → 읍면동이 위계적으로 함께 포함되어 있어,
  단순 합산·평균 시 상위 집계 행과 최하위 행정동이 섞여 통계가 왜곡될 수 있습니다.
- CSV의 숫자 컬럼은 천단위 콤보(,)가 포함된 문자열이므로, 앱 내부에서 콤마 제거 후
  숫자형으로 변환하는 전처리를 거칩니다. 인코딩은 CP949입니다.
- 유사 지역 탐색 시에는 지역 규모(총인구수) 차이를 제거하기 위해 연령대별 인구를
  **총인구 대비 비율**로 정규화한 뒤 유클리드 거리를 계산합니다. 이로 인해 규모가 크게
  다른 지역(예: 특정 읍면동과 특정 시 전체)도 인구구조 형태가 비슷하면 유사 지역으로
  탐색될 수 있습니다.
